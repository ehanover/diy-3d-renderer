# DIY-OpenGL

My exploration of 3D rendering. 

```
sudo apt install libsdl2-2.0-0 libsdl2-dev

git clone https://github.com/ehanover/diy-opengl.git
cd diy-opengl
make
make install
./diy-opengl
```
